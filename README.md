Primeiros Projetos em Java

Este repositório reúne meus primeiros projetos e exercícios práticos em Java, desenvolvidos durante meus estudos iniciais em programação.

O objetivo principal foi consolidar fundamentos, lógica de programação e conceitos essenciais da linguagem Java.

---

Conceitos trabalhados
- Lógica de programação
- Estruturas condicionais
- Laços de repetição
- Vetores e matrizes
- Estruturas de dados básicas
- Organização de código em classes

---

Estrutura do repositório

Cada pasta representa um exercício ou mini projeto independente:

- `CalculadoraSimples` – Operações matemáticas básicas
- `JogoDosNumeros` – Jogo de adivinhação com números aleatórios
- `ParOuImpar` – Verificação de números pares e ímpares
- `EstruturaDeDados` – Exercícios com vetores e listas
- `SistemaDeGestao` – Simulação simples de sistema em Java

---

Objetivo do repositório
- Registrar minha evolução como desenvolvedor
- Praticar conceitos fundamentais
- Servir como base para refatorações futuras
